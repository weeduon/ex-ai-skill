# 隐私说明

前任AIskill 的开源版本默认采用本地优先模式。

## 本地保存内容

默认目录：

```txt
~/.ex-ai-skill/
```

其中可能包含：

- 测试码激活状态的哈希
- 用户配置的大模型 Provider
- 用户上传的聊天记录解析结果
- 角色人格包
- 本地对话记录
- 本地日志

## API Key

用户自己的 API Key 默认保存在本地。请不要把真实 API Key 提交到 GitHub，也不要分享给其他人。

## 聊天记录

聊天记录默认仅用于本地解析和角色人格包生成。用户应确保自己有权处理相关聊天记录。

## 删除数据

可以直接删除本地目录来清空全部数据：

```bash
rm -rf ~/.ex-ai-skill
```

Windows 用户可删除用户目录下的 `.ex-ai-skill` 文件夹。
